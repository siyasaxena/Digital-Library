# Digital Library Management System

A full-stack web application built to manage a personal collection of books. This project demonstrates RESTful API principles and CRUD (Create, Read, Update, Delete) operations using the Node.js ecosystem.

## Features
* Add New Books: Input title, author, and image URLs to your collection.
* View Library: A clean, responsive dashboard showing all saved books using EJS templates.
* Update Details: Edit book information dynamically.
* Delete Records: Remove books from the library with a single click.
* Responsive Design: Styled with custom CSS and Bootstrap for a modern look.

## Tech Stack
* Frontend: EJS (Embedded JavaScript), HTML5, CSS3 (Flexbox/Grid), Bootstrap
* Backend: Node.js, Express.js
* Version Control: Git and GitHub
